# ❤️ Heart Disease Prediction Using Machine Learning

## Project Overview
This project predicts the presence of heart disease using Machine Learning algorithms based on patient medical data.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
